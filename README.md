# Glow_calculator
🧮 Glow Calculator  A sleek, modern calculator built with HTML, CSS, and vanilla JavaScript, featuring a glowing dark UI, animated background orbs, ripple click effects, and smooth button animations.
✨ Features


Basic arithmetic operations — addition, subtraction, multiplication, and division
Expression preview — shows the full expression above the result
Ripple effect — material-style ripple animation on every button press
Pop animation — buttons subtly bounce on click
Animated background orbs — soft glowing gradients floating in the background
Responsive, dark-themed UI with a glassmorphism-style calculator card


📂 Project Structure

├── index.html    # Markup structure of the calculator
├── style1.css    # Styling, animations, and layout
└── calc.js       # Calculator logic and interactivity

🚀 How to Run


Clone or download this repository.
Open index.html in any modern web browser.
Start calculating! 🎉


No build tools, dependencies, or installations required — it's pure HTML, CSS, and JS.

🛠️ Tech Stack


HTML5 — structure
CSS3 — styling, animations, gradients, glassmorphism effects
JavaScript (Vanilla) — calculation logic, event handling, animations


🧩 How It Works


Numbers and operators are appended to an expression string as you click buttons.
Pressing = evaluates the expression and displays the result, while the original expression is shown above for reference.
Pressing C clears the display and resets the calculator.
Errors (like division by zero or invalid expressions) are handled gracefully with an Error or Syntax Error message.
